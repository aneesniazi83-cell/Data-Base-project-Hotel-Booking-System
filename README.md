# Hotel Booking System

A full-stack Hotel Booking Management System developed using **PHP, MySQL, HTML, CSS, and JavaScript**. This project was designed to manage hotel operations efficiently, including room bookings, customer records, staff management, hotel services, and check-in/check-out activities.

## Features

* Room booking and reservation management
* Customer registration and record management
* Staff management module
* Check-in and check-out system
* Hotel services management
* CRUD operations for all modules
* Database integration using MySQL
* User-friendly frontend interface

## Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: PHP
* Database: MySQL
* Server Environment: XAMPP
* Database Management Tool: phpMyAdmin

## Database Design

The project includes a complete relational database designed through an **Entity Relationship Diagram (ERD)**. Multiple modules and relationships were analyzed and implemented to maintain proper data management and normalization.

## Purpose

This project was developed for learning and implementing concepts of:

* Database Management Systems (DBMS)
* Full-stack web development
* Relational database design
* Backend and frontend integration
* CRUD operations and data handling

## Setup Instructions

1. Install XAMPP and start Apache & MySQL.
2. Import the SQL database file into phpMyAdmin.
3. Place the project folder inside the `htdocs` directory.
4. Open `localhost/project-folder-name` in the browser.
